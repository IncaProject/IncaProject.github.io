NAME

inca-null-reporter

DESCRIPTION

This is an Inca reporter that takes all information necessary to produce a report from command line arguments. If desired, a body for the report can be fed via stdin.

SYNOPSIS

inca-null-reporter [options]

OPTIONS

--

indicates that any subsequent arguments should be listed as args in the report.

--error_message

the exit status error message

--gmt

the report time stamp; the script will translate a local epoch time to ISO 8601 format if necessary

--hostname

the report host name

--name

the reporter name

--completed

the exit status completed value

--version

the reporter version
