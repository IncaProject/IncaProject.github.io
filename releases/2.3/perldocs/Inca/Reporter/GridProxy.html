NAME

Inca::Reporter::GridProxy - A pseudo-module indicating that a reporter requires a proxy credential in order to execute.

SYNOPSIS

  my $reporter = new Inca::Reporter();
  $reporter->addDependency('Inca::Reporter::GridProxy');

DESCRIPTION

Declaring a dependency on this module is a signal to Inca's ReporterManager that a grid proxy credential needs to be initialized before the reporter can be run.
